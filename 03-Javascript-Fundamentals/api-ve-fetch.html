Ders 12: İnternetten Veri Çekmek

Dış Dünyaya Açılıyoruz: API ve Fetch Kavramı

1. API Nedir? (Application Programming Interface)

Kendi yazdığımız kodlar harika ama ya güncel hava durumuna, döviz kurlarına veya filmlerin IMDB puanlarına ihtiyacımız olursa? Bunları tek tek biz yazamayız. Başka sunucuların (bilgisayarların) bize bu verileri sağlaması gerekir. İşte iki yazılımın birbiriyle konuşmasını sağlayan bu köprüye API denir.

Kafamızda Canlandıralım: Restorandaki Garson 🍽️
Siz masada oturan müşterisiniz (Web Sayfası). Mutfakta ise şef var (Veritabanı/Sunucu). Siz mutfağa gidip kendi yemeğinizi alamazsınız. Siparişinizi garsona söylersiniz, garson mutfağa gider, yemeği alır ve size getirir. İşte internet dünyasındaki Garson = API'dir!

2. Asenkron Ne Demek? (Beklemeyi Öğrenmek)

JavaScript çok hızlıdır, kodları yukarıdan aşağıya saniyeler içinde okur. Ancak internetten bir veri istediğimizde, bu verinin gelmesi kullanıcının internet hızına göre 1-2 saniye sürebilir.

Kargo Beklerken Hayat Durmaz 📦
İnternetten bir kargo sipariş ettiğinizde, kargo gelene kadar kapının önünde dikilip beklemezsiniz. Hayatınıza devam edersiniz, kargo gelince alırsınız. JavaScript de böyledir. İnternetten veri isterken Asenkron (eşzamanlı olmayan) bir yapı kurmalıyız ki, veri gelene kadar sitemiz donup kalmasın.

3. Siparişi Vermek: Fetch Kavramı

JavaScript'te garsonu çağırmanın ve sipariş vermenin kodu fetch() komutudur. Modern ES6 standartlarıyla bunu yapmanın en şık yolu async / await (asenkron / bekle) kullanmaktır.

// Fonksiyonumuzun başına 'async' ekliyoruz, "bu işlemin içinde biraz bekleyeceğiz" diyoruz.
const veriGetir = async () => {
    
    // 1. Garsona siparişi ver ve yemeğin gelmesini BEKLE (await)
    const cevap = await fetch("https://api.ornek-site.com/veriler");
    
    // 2. Gelen cevabı bizim okuyabileceğimiz JSON (Metin) formatına çevirmesini BEKLE
    const veri = await cevap.json();
    
    // 3. Veriyi ekrana yazdır
    console.log(veri);
}

4. Mini Proje: Rastgele Köpek Fotoğrafı Uygulaması 🐶

Şimdi bunu gerçek bir API ile test edelim! İnternette herkesin kullanımına açık, bize her çağırdığımızda rastgele bir köpek fotoğrafı adresi gönderen bir API var. Hadi onu sayfamıza ekleyelim.

<!-- HTML Kısmı -->
<button id="kopekGetirBtn">Yeni Köpek Getir!</button>
<br><br>
<img id="kopekResmi" src="" width="300" style="border-radius: 10px;">

// JavaScript Kısmı
const buton = document.getElementById("kopekGetirBtn");
const resim = document.getElementById("kopekResmi");

// Butona tıklanınca çalışacak asenkron fonksiyon
buton.addEventListener("click", async () => {
    
    // Kullanıcıya beklediğini hissettirmek için yükleniyor yazabiliriz
    buton.innerHTML = "Yükleniyor... ⏳";

    // API'den veriyi çek
    const cevap = await fetch("https://dog.ceo/api/breeds/image/random");
    const sonuc = await cevap.json();

    // Gelen veri içindeki 'message' kısmı fotoğrafın linkini barındırır.
    // Bu linki HTML'deki img etiketimizin src (kaynak) özelliğine ekliyoruz.
    resim.src = sonuc.message;
    
    // İşlem bitince butonu eski haline çevir
    buton.innerHTML = "Yeni Köpek Getir!";
    
});
