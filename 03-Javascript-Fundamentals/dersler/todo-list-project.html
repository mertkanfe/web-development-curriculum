Ders 8: Kodlama Atölyesi

Öğrendiklerimizi Birleştiriyoruz: To-Do List (Yapılacaklar Listesi) Projesi

1. Projemizin Amacı Nedir?

Bu derste, şu ana kadar öğrendiğimiz Değişkenler, Fonksiyonlar ve DOM Manipülasyonu konularını bir araya getirerek gerçek bir uygulama yapacağız. Kullanıcı bir metin kutusuna görev yazacak, "Ekle" butonuna basacak ve bu görev aşağıdaki listede belirecek!

Adım Adım Düşünelim (Algoritma Mantığı):
1. Kullanıcının yazı yazdığı kutuyu ve butonu bul.
2. Butona tıklandığında (Event), kutunun içindeki yazıyı al.
3. Yeni bir liste elemanı (li) oluştur ve yazıyı içine koy.
4. Bu yeni elemanı sayfa üzerindeki mevcut listeye (ul) ekle.
5. Kutunun içini bir sonraki görev için temizle.

2. Sahneyi Hazırlayalım: HTML İskeleti

Önce kuklalarımızı (HTML etiketlerimizi) sahneye yerleştirelim ki JavaScript ile iplerini çekebilelim.

<!-- Bu kodları kendi HTML dosyanızın body kısmına yazın -->
<div class="uygulama-kutusu">
    <h2>Günlük Görevlerim 📝</h2>
    
    <!-- Kullanıcıdan veri alacağımız input ve buton -->
    <input type="text" id="gorevGirdisi" placeholder="Yeni bir görev yazın...">
    <button id="ekleButonu">Listeye Ekle</button>

    <!-- Görevlerin sıralanacağı boş listemiz -->
    <ul id="gorevListesi">
        <!-- Yeni görevler buraya gelecek -->
    </ul>
</div>

3. Sihri Başlatalım: JavaScript Kodları

Şimdi app.js dosyamıza geçelim ve sırasıyla algoritmamızı koda dökelim.

Yeni Bir Komut Öğreniyoruz: createElement()
HTML dosyamızda olmayan yepyeni bir etiketi JavaScript ile yoktan var etmek için document.createElement("etiket-adi") kullanırız. 
// 1. ADIM: İhtiyacımız olan tüm HTML elemanlarını (kuklaları) yakalayalım
const gorevKutusu = document.getElementById("gorevGirdisi");
const ekleBtn = document.getElementById("ekleButonu");
const liste = document.getElementById("gorevListesi");

// 2. ADIM: Butona tıklanma olayını (Event) dinleyelim
ekleBtn.addEventListener("click", function() {
    
    // Kutunun içindeki yazıyı alıyoruz
    let yeniGorev = gorevKutusu.value;
    
    // Eğer kutu boşsa uyarı ver ve işlemi durdur (if kontrolü!)
    if (yeniGorev === "") {
        alert("Lütfen boş bir görev eklemeyin!");
        return; // Fonksiyonu burada keser, aşağıya inmez
    }

    // 3. ADIM: Yeni bir liste elemanı (<li>) oluşturalım
    let yeniMadde = document.createElement("li");
    
    // İçine kullanıcının yazdığı metni ekleyelim
    yeniMadde.innerHTML = yeniGorev;

    // 4. ADIM: Bu yeni maddeyi, sayfadaki listemizin (ul) içine yapıştıralım (appendChild)
    liste.appendChild(yeniMadde);

    // 5. ADIM: Yeni bir şey yazabilmek için kutunun içini temizleyelim
    gorevKutusu.value = "";
    
});
appendChild() Nedir?
İngilizce "Çocuk Ekle" anlamına gelir. Bir HTML etiketinin içine (sonuna) yeni bir etiket yerleştirmek için kullanırız. Bizim örneğimizde <ul> ebeveyn, <li> ise eklenecek çocuktur!
