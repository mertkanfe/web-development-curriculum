2. Senaryo 1: En Büyük Sayıyı Bulmak (Kralı Seçmek)

Elimizde bir sınıfın sınav notları var. Amacımız sınıftaki en yüksek notu bulan bir program yazmak.

Kafamızda Canlandıralım: Sınıfın En Uzun Boylu Öğrencisi 📏
Sınıftaki en uzun kişiyi nasıl bulursunuz? Gözünüze ilk çarpan kişiyi "Şimdilik En Uzun" olarak seçersiniz. Sonra onu sınıftaki diğer herkesle tek tek yan yana getirirsiniz. Eğer karşısına ondan daha uzun biri çıkarsa, artık yeni "En Uzun" o kişi olur. Liste bitene kadar bu düelloyu yaparsınız. Ayakta kalan son kişi sınıfın en uzunudur! 
// 1. Notlarımızı bir diziye koyalım
int[] notlar = {45, 82, 67, 95, 33, 78};

// 2. İlk sıradaki notu geçici olarak "Kral" (En Büyük) ilan edelim
int enBuyukNot = notlar[0]; 

// 3. Döngü ile diğer tüm notlara sırayla bakalım (1. indexten başlıyoruz çünkü 0'ı zaten kral seçtik)
for (int i = 1; i < notlar.length; i++) {
    
    // Eğer sıradaki not, bizim şu anki kralımızdan büyükse...
    if (notlar[i] > enBuyukNot) {
        
        // Tahtı devret! Artık yeni kral o.
        enBuyukNot = notlar[i];
    }
}

// Döngü bittiğinde enBuyukNot değişkeninin içinde gerçekten en büyük sayı kalmış olacak
System.out.println("Sınıfın en yüksek notu: " + enBuyukNot);

3. Senaryo 2: Doğrusal Arama (Linear Search)

Şimdi de 500 kişilik bir müşteri listesinde "Ahmet" isimli kişinin listede olup olmadığını, varsa kaçıncı sırada olduğunu bulalım.

Kafamızda Canlandıralım: Kalabalıkta Arkadaşını Aramak 🕵️‍♂️
Bir sıraya dizilmiş insanlara tek tek "Sen Ahmet misin?", "Sen Ahmet misin?" diye sorarak ilerlersiniz. Ahmet'i bulduğunuz an aramayı durdurursunuz. Eğer kuyruğun sonuna gelirseniz ve kimse "Evet" demezse, Ahmet orada yoktur. 
String[] musteriler = {"Ayşe", "Mehmet", "Elif", "Ahmet", "Zeynep"};
String arananKisi = "Ahmet";
boolean bulunduMu = false; // Başlangıçta yok varsayıyoruz

for (int i = 0; i < musteriler.length; i++) {
    
    // Sıradaki kişi aradığımız kişi mi?
    if (musteriler[i].equals(arananKisi)) {
        System.out.println(arananKisi + " bulundu! Sıra numarası: " + i);
        bulunduMu = true;
        
        // Bulduğumuz için aramaya devam etmeye gerek yok, döngüyü KIRIYORUZ (break)
        break; 
    }
}

// Eğer döngü bittiğinde bulunduMu hala false ise, listede yoktur.
if (bulunduMu == false) {
    System.out.println("Maalesef " + arananKisi + " listede yok.");
}
Sihirli Kelime: "break" 🛑
Döngüleri erkenden durdurmak için break komutunu kullanırız. 1000 kişilik listede aradığımız kişiyi 3. sırada bulduysak, boşuna 997 kişiye daha sormak bilgisayarı yorar. break bizi bu yükten kurtarır!
