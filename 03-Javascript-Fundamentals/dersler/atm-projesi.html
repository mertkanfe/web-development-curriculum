1. Projemizin Amacı

Bu projede OOP (Nesne Yönelimli Programlama) ve algoritmaları bir araya getireceğiz. Kullanıcıya siyah ekranda (konsolda) bir menü sunacağız. Kullanıcı klavyeden seçim yapacak ve sistemimiz kapsülleme (encapsulation) kurallarına uygun olarak hesap bakiyesini güncelleyecek.

2. Yeni Bir Kavram: Kullanıcıdan Veri Almak (Scanner)

Frontend bölümünde kullanıcıdan veriyi HTML'deki <input> kutularından alıyorduk. Ancak saf bir arka plan (Java/Backend) uygulamasında klavyeden yazılanları okumak için özel bir alete ihtiyacımız var.

Scanner (Tarayıcı) Sınıfı 📡
Bilgisayarın klavyesini dinleyen bir kulaktır. Scanner klavye = new Scanner(System.in); diyerek bu kulağı aktif ederiz. Daha sonra klavye.nextInt(); dediğimizde program durur ve kullanıcının bir sayı yazıp Enter'a basmasını bekler.

3. Sonsuz Menü: "While" Döngüsü

ATM'ler siz "Kartı İade Et" tuşuna basana kadar kapanmaz, sürekli size ana menüyü gösterir. Bunu sağlamak için While (Sürece) döngüsünü kullanırız.

Kafamızda Canlandıralım: Bekçi Köpeği 🐕
While döngüsü "şart doğru olduğu SÜRECE içindeki kodları tekrarla" demektir. while(uygulamaCalisiyorMu == true) dersek, biz o değişkeni "false" yapana kadar menü ekranda dönüp duracaktır.

4. Proje Kodları (Tüm Parçalar Birleşiyor)

Aşağıdaki kodu inceleyin. Önce Kapsüllenmiş bir Sınıf (BankaHesabi), ardından da onu kontrol eden sonsuz bir döngü menüsü göreceksiniz.

import java.util.Scanner; // Klavye dinleyicisini projeye dahil ettik

// 1. ADIM: NESNEMİZİ VE KURALLARINI OLUŞTURALIM
class BankaHesabi {
    private int bakiye = 1000; // Kapsülleme (Dışarıdan doğrudan ulaşılamaz)

    public int getBakiye() {
        return bakiye;
    }

    public void paraYatir(int miktar) {
        if(miktar > 0) {
            bakiye = bakiye + miktar;
            System.out.println(miktar + " TL hesabınıza eklendi.");
        }
    }

    public void paraCek(int miktar) {
        if(miktar <= bakiye && miktar > 0) {
            bakiye = bakiye - miktar;
            System.out.println(miktar + " TL çektiniz.");
        } else {
            System.out.println("Hata: Yetersiz bakiye veya geçersiz işlem!");
        }
    }
}

// 2. ADIM: ANA PROGRAM (ATM MENÜSÜ)
public class AnaProgram {
    public static void main(String[] args) {
        
        Scanner klavye = new Scanner(System.in);
        BankaHesabi hesap = new BankaHesabi(); // Müşterimizin hesabı oluştu
        
        boolean sistemAcik = true;

        System.out.println("--- KODLAMA BANKASINA HOŞ GELDİNİZ ---");

        // Sonsuz ATM Döngüsü
        while(sistemAcik) {
            System.out.println("\n1- Bakiye Sorgula");
            System.out.println("2- Para Yatır");
            System.out.println("3- Para Çek");
            System.out.println("4- Çıkış Yap");
            System.out.print("Seçiminiz: ");
            
            int secim = klavye.nextInt(); // Kullanıcıdan sayı al

            if(secim == 1) {
                System.out.println("Mevcut Bakiyeniz: " + hesap.getBakiye() + " TL");
            } 
            else if(secim == 2) {
                System.out.print("Yatırılacak tutar: ");
                int tutar = klavye.nextInt();
                hesap.paraYatir(tutar);
            } 
            else if(secim == 3) {
                System.out.print("Çekilecek tutar: ");
                int tutar = klavye.nextInt();
                hesap.paraCek(tutar);
            } 
            else if(secim == 4) {
                System.out.println("Kartınız iade ediliyor. İyi günler!");
                sistemAcik = false; // Döngüyü kırar ve programı bitirir
            } 
            else {
                System.out.println("Lütfen 1 ile 4 arasında geçerli bir tuşa basın.");
            }
        }
    }
}
Switch - Case Kullanımı
Yukarıdaki uzun if - else if zinciri yerine, menü tasarımlarında genellikle switch (secim) { case 1: ... break; } yapısı da tercih edilebilir. İkisi de aynı işi yapar, hangisi kolayınıza geliyorsa onu seçin!
