Ders 11: JavaScript'in Modern Yüzü

Daha Kısa, Daha Şık Kodlar: ES6 Standartları

1. ES6 Nedir? (Yazılımın Evrimi)

Diller zamanla değişir. Nasıl ki günlük hayatta eskisi gibi divan edebiyatı kelimeleriyle konuşmuyorsak, JavaScript de zamanla modernleşmiştir. 2015 yılında çıkan ES6 (ECMAScript 6) güncellemesi, dilin yapısını tamamen değiştirerek kod yazmayı çok daha pratik hale getirdi.

Bugün React, Vue veya Node.js gibi teknolojileri öğrenmek istiyorsanız, bu modern yazım tarzını bilmek zorundasınız.

2. Artı (+) Çilesine Son: Template Literals

Eskiden değişkenleri ve metinleri birleştirirken sürekli artı (+) işareti kullanmak ve boşlukları ayarlamak bir kabustu.

Kafamızda Canlandıralım: Boşluk Doldurmaca 📝
Cümleleri bantla (artı işaretiyle) birbirine yapıştırmak yerine, yeni sistemde sadece bir cümle kalıbı yazarız ve değişkenlerin geleceği yerlere ${ } ile boşluklar açarız.
Önemli Kısayol: Backtick ( ` ) İşareti
Bu sihirli özelliği kullanmak için normal tırnak yerine "Backtick" (Ters Tırnak) kullanmalıyız. Türkçe Windows klavyelerde AltGr + Noktalı Virgül (;) tuşlarına aynı anda basarak yapabilirsiniz. 
let ad = "Mert";
let yas = 26;

// ❌ ESKİ YÖNTEM (Hata yapması çok kolay)
let eskiMesaj = "Merhaba benim adım " + ad + " ve ben " + yas + " yaşındayım.";

// ✅ YENİ YÖNTEM (Template Literals - Temiz ve okunaklı)
let yeniMesaj = `Merhaba benim adım ${ad} ve ben ${yas} yaşındayım.`;

3. Tek Satırda Mantık: Ternary Operator

Bazen 5 satırlık bir `if-else` bloğu yazmak yerine, bu işlemi tek satırda çözmek isteriz. Buna Ternary (Üçlü) Operatör denir. Formülü basittir: Koşul ? Doğruysa : Yanlışsa

let puan = 85;

// ❌ ESKİ YÖNTEM (Çok fazla satır)
let sonuc;
if (puan >= 50) {
    sonuc = "Geçtin";
} else {
    sonuc = "Kaldın";
}

// ✅ YENİ YÖNTEM (Sadece tek satır!)
let yeniSonuc = puan >= 50 ? "Geçtin" : "Kaldın";

4. Fonksiyonların Yeni Evrimi: Arrow Functions (Ok Fonksiyonları)

JavaScript'te function kelimesini yazmaktan sıkılmadınız mı? Arrow Functions sayesinde daha kısa ve modern fonksiyonlar yazabiliyoruz.

// ❌ ESKİ YÖNTEM
function topla(a, b) {
    return a + b;
}

// ✅ YENİ YÖNTEM (Arrow Function)
const topla = (a, b) => {
    return a + b;
}

// 🚀 DAHA DA KISASI! (Eğer fonksiyon tek satırsa return'e ve süslü paranteze gerek yoktur)
const kisaTopla = (a, b) => a + b;

5. Kutuları Kolayca Açmak: Destructuring

Dizilerin içindeki verilere tek tek indeks numarasıyla (dizi[0], dizi[1]) erişmek yerine, onlara doğrudan isim vererek kutudan çıkarabiliriz.

let renkler = ["Kırmızı", "Mavi", "Yeşil"];

// Eski yöntem
let ilkRenk = renkler[0];
let ikinciRenk = renkler[1];

// ✅ YENİ YÖNTEM (Destructuring)
// Dizideki 1. elemanı 'renk1'e, 2. elemanı 'renk2'ye ata.
const [renk1, renk2] = renkler;

console.log(renk1); // Çıktı: Kırmızı

Modern Geliştiriciler Kulübüne Hoş Geldiniz! 🎯

Gördüğünüz gibi modern JavaScript, daha az kod yazarak daha çok iş yapmamızı sağlıyor. Bundan sonraki projelerimizde `function` kelimesi yerine okları (=>), `+` işaretleri yerine ise backtick (`) işaretlerini kullanmaya alışacağız.
