Ders 7: Web Sayfasına Hükmetmek

JavaScript ile HTML'i Canlandırmak: DOM ve Etkileşim

1. DOM (Document Object Model) Nedir?

Tarayıcınız bir HTML sayfasını yüklediğinde, arka planda bu sayfanın bir haritasını veya iskeletini çıkarır. Bu haritaya DOM (Belge Nesne Modeli) denir.

Kafamızda Canlandıralım: Kukla ve İpleri 🎭
Web sayfanızdaki her bir HTML etiketini (başlıklar, butonlar, resimler) bir kukla gibi düşünün. DOM, bu kuklalara bağlanan iplerdir. JavaScript ise bu ipleri tutan kuklacıdır! JS kullanarak ipleri çeker, butonların rengini değiştirir veya ekrana yeni yazılar ekleriz.

2. Kuklanın İplerini Yakalamak (Eleman Seçimi)

JavaScript ile bir HTML elemanını değiştirmeden önce, onu "yakalamamız" (seçmemiz) gerekir. Bunun için document kelimesini kullanırız.

<!-- HTML Kısmı -->
<h1 id="ana-baslik">Eski Başlık</h1>
<button class="kirmizi-buton">Tıkla</button>

// JavaScript Kısmı
// 1. ID'sine göre yakalamak (En çok kullanılan!)
let baslik = document.getElementById("ana-baslik");

// 2. CSS Seçicisi gibi yakalamak (Modern yöntem)
let buton = document.querySelector(".kirmizi-buton");

3. Sayfayı Değiştirmek (Manipülasyon)

Elemanı yakaladıktan sonra onunla istediğimiz gibi oynayabiliriz. İçindeki yazıyı, HTML yapısını veya CSS stilini anında değiştirebiliriz.

let mesajKutusu = document.getElementById("mesaj");

// İçindeki yazıyı değiştirmek
mesajKutusu.innerHTML = "Sisteme Başarıyla Giriş Yaptınız! ✅";

// CSS özelliklerini değiştirmek
mesajKutusu.style.color = "white";
mesajKutusu.style.backgroundColor = "green";
mesajKutusu.style.padding = "20px";

4. Sihrin Başladığı Yer: Olay Dinleyicileri (Events)

JavaScript, kullanıcıların sitede ne yaptığını sürekli izleyebilir. Bir butona tıklandığında, fare bir resmin üzerine geldiğinde veya klavyede bir tuşa basıldığında anında tepki verebiliriz. Buna Olay (Event) denir.

addEventListener() Nedir?
Kelime anlamı "Olay Dinleyicisi Ekle"dir. JavaScript'e "Şu elemanı dinle, eğer belirttiğim olay (örneğin tıklama) gerçekleşirse şu fonksiyonu çalıştır" demektir.

Örnek: Karanlık Mod Butonu Yapalım 🌙

<!-- HTML Butonumuz -->
<button id="temaButonu">Karanlık Moda Geç</button>

// JavaScript Kodumuz
let temaBtn = document.getElementById("temaButonu");

// Butona tıklanma (click) olayını dinliyoruz
temaBtn.addEventListener("click", function() {
    
    // Sayfanın arka planını siyaha, yazıları beyaza çevir
    document.body.style.backgroundColor = "black";
    document.body.style.color = "white";
    
    // Butonun içindeki yazıyı değiştir
    temaBtn.innerHTML = "Aydınlık Moda Geç ☀️";
    
});

5. Formlardan Veri Almak (Kullanıcı Girdisi)

Öğrencilerin en çok kullanacağı şeylerden biri, kullanıcının girdiği metinleri okumaktır. Bir <input> (girdi) kutusunun içindeki değere ulaşmak için .value özelliğini kullanırız.

<!-- HTML Girdi Kutusu -->
<input type="text" id="kullaniciAdi" placeholder="Adınızı girin">

// JavaScript ile kutunun içindeki yazıyı okumak
let isimKutusu = document.getElementById("kullaniciAdi");
console.log("Hoş geldin, " + isimKutusu.value);
