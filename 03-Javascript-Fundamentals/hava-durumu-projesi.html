Gerçek Verilerle Çalışmak: Hava Durumu Uygulaması 🌤️

1. Projemizin Amacı Nedir?

Bu atölyede, kullanıcıdan bir şehir ismi alacağız, bu şehir ismini hava durumu sağlayan bir sunucuya (API) göndereceğiz ve gelen gerçek sıcaklık verisini ekrana yazdıracağız. Bu projede ES6 özelliklerini (Arrow Functions, Template Literals) aktif olarak kullanacağız.

Yeni Bir Kavram: API Key (Kimlik Kartı) 💳
Hava durumu sunucuları (örneğin OpenWeatherMap), kapılarını herkese sonuna kadar açmaz. Çok fazla istek gelip sunucunun çökmesini engellemek için, kayıt olan herkese özel bir şifre (API Key) verir. Biz de siparişimizi verirken garsona "Al bu da benim kimliğim" diyerek bu şifreyi göstereceğiz.

2. Sahneyi Hazırlayalım: HTML İskeleti

Kullanıcının şehir ismini girebileceği bir kutu ve sonuçları göstereceğimiz bir alan oluşturuyoruz.

<!-- Uygulama Kutusu -->
<div class="hava-durumu-kutusu">
    <h2>Anlık Hava Durumu</h2>
    
    <input type="text" id="sehirGirdisi" placeholder="Örn: Istanbul">
    <button id="sorgulaButonu">Hava Durumunu Getir</button>

    <!-- Veriler Geldiğinde Burası Dolacak -->
    <div id="sonucAlani" style="margin-top: 20px;">
        <!-- <h3>Istanbul</h3> -->
        <!-- <p>Sıcaklık: 24°C</p> -->
    </div>
</div>

3. JavaScript Kodlarımız (Sihrin Gerçekleştiği Yer)

Aşağıdaki kodda fetch ile asenkron (beklemeli) bir işlem yapacağız. Kodların ES6 standartlarıyla (const, let, =>, `${}`) yazıldığına dikkat edin!

Not: Gerçek bir projede OpenWeatherMap sitesinden kendi API Key'inizi almanız gerekir. Biz bu derste eğitim amaçlı örnek bir API linki kullanacağız. 
// 1. HTML Elemanlarını Yakalayalım
const sehirGirdisi = document.getElementById("sehirGirdisi");
const sorgulaBtn = document.getElementById("sorgulaButonu");
const sonucAlani = document.getElementById("sonucAlani");

// (Temsili API Anahtarımız)
const API_KEY = "sizin_gizli_anahtariniz"; 

// 2. Butona Tıklanma Olayı (Arrow Function ve Async kullanıyoruz)
sorgulaBtn.addEventListener("click", async () => {
    
    const sehir = sehirGirdisi.value;

    // Boş gönderimi engelle
    if (sehir === "") {
        alert("Lütfen bir şehir adı girin!");
        return;
    }

    // 3. Yükleniyor mesajı gösterelim
    sonucAlani.innerHTML = "<p>Hava durumu sorgulanıyor... ⏳</p>";

    try {
        // 4. API'den Veriyi Çekelim (Template Literals ile linki birleştiriyoruz)
        // Not: Bu link örnektir. Gerçek hayatta linkin sonuna &appid=${API_KEY} eklenir.
        const cevap = await fetch(`https://api.openweathermap.org/data/2.5/weather?q=${sehir}&units=metric&appid=${API_KEY}`);
        
        // Gelen cevabı JSON formatına çevir (Bekle)
        const veri = await cevap.json();

        // Eğer şehir bulunamazsa (Örn: 404 hatası)
        if (veri.cod === "404") {
            sonucAlani.innerHTML = "<p style='color:red;'>Şehir bulunamadı! Harf hatası yapmış olabilirsiniz.</p>";
            return;
        }

        // 5. Gelen Veriyi Ekrana Yazdıralım
        // Gelen JSON dosyasının içindeki name (şehir adı) ve main.temp (sıcaklık) verilerini alıyoruz
        sonucAlani.innerHTML = `
            <h3>📍 ${veri.name}, ${veri.sys.country}</h3>
            <h1 style="color: #ed8936;">${veri.main.temp} °C</h1>
            <p>Durum: ${veri.weather[0].description}</p>
        `;

    } catch (hata) {
        // İnternet kopması gibi beklenmeyen hataları yakalarız
        sonucAlani.innerHTML = "<p>Bağlantı hatası oluştu.</p>";
        console.log("Hata detayı: ", hata);
    }
    
});
Try / Catch Bloğu Nedir?
Asenkron işlemlerde sunucu çökebilir veya kullanıcının interneti kesilebilir. try (dene) kısmına kodlarımızı yazarız. Eğer bir çökme olursa program hata verip durmak yerine, kontrolü gizlice catch (yakala) kısmına geçirir ve kullanıcıya kibar bir hata mesajı gösteririz.
