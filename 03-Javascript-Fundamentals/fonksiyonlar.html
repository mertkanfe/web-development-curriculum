Ders 6: Kendi Komutlarımızı Oluşturma

Kod Yazmayı Kolaylaştıran Paketler: Fonksiyonlar (Functions)

1. Fonksiyon Nedir?

Fonksiyonlar, belirli bir işi yapmak için gruplandırılmış kod bloklarıdır. Bir fonksiyonu bir kez tanımlarsınız ve ihtiyacınız olan her yerde ismini söyleyerek (çağırarak) çalıştırırsınız.

Kafamızda Canlandıralım: Mutfak Robotu 🍹
Bir mutfak robotunu düşünün. İçine meyveleri koyarsınız, düğmesine basarsınız ve o size bir içecek hazırlar. Robotun içindeki bıçakların nasıl döndüğünü her seferinde yeniden tasarlamanıza gerek yoktur. Sadece meyveleri (girdi) verirsiniz ve sonucu (çıktı) alırsınız. JavaScript'te fonksiyonlar tam olarak budur!

2. Fonksiyon Nasıl Yazılır? (Tanımlama)

Bir fonksiyon oluşturmak için function anahtar kelimesini kullanırız, ardından fonksiyona bir isim veririz.

// 1. Fonksiyonu oluşturuyoruz (Tanımlama)
function selamVer() {
    console.log("Merhaba arkadaşlar!");
    console.log("Bugün JavaScript öğreniyoruz.");
}

// 2. Fonksiyonu çalıştırma (Çağırma)
selamVer(); 
selamVer(); // İstediğimiz kadar çağırabiliriz!

3. Parametreler: Fonksiyona Bilgi Göndermek

Bazen fonksiyonlarımızın dışarıdan bilgiye ihtiyacı olur. Örneğin "Selam ver" demek yerine "Mert'e selam ver" demek istiyorsak, fonksiyonun parantez içine bir değişken (parametre) yazarız.

function kisiyeSelamVer(isim) {
    console.log("Merhaba " + isim + ", hoş geldin!");
}

// Fonksiyonu çağırırken bilgiyi gönderiyoruz (Argüman)
kisiyeSelamVer("Mert"); 
kisiyeSelamVer("Zeynep");

4. Return: Sonucu Geri Almak

Bazı fonksiyonlar sadece bir iş yapmaz, bize bir sonuç üretir. Mutfak robotunun bize meyve suyunu uzatması gibi, fonksiyon da return kelimesiyle bize bir değer döndürür.

function toplamaYap(sayi1, sayi2) {
    let sonuc = sayi1 + sayi2;
    return sonuc; // Sonucu dışarıya fırlat!
}

// Dönen sonucu bir değişkene kaydedebiliriz
let toplam = toplamaYap(10, 20);
console.log("Sonuç: " + toplam); // Çıktı: 30
Neden Return Kullanırız?
Eğer bir fonksiyonun ürettiği cevabı başka bir işlemde (örneğin bir indirim hesaplamasında) kullanmak istiyorsak return şarttır. console.log sadece ekrana yazar, ama return o veriyi elimize verir.

5. Neden Fonksiyon Kullanmalıyız?

  • Tekrarı Önler: Aynı kodu defalarca yazmazsınız.
  • Düzen Sağlar: Kodunuzu küçük, anlaşılır parçalara böler.
  • Kolay Hata Bulma: Bir işlemde hata varsa, sadece o fonksiyonun içine bakmanız yeterlidir.
  • Okunabilirlik: kullaniciyiKaydet() gibi isimler, kodun ne yaptığını şiir gibi anlatır.
