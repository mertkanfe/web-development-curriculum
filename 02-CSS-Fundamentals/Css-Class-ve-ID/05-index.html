CSS3 | CSS ile Class ve ID Kullanımı #5

CSS'te class (sınıf) kullanımı, HTML elementlerini gruplayarak aynı tasarım özelliklerini birden fazla elemente aynı anda ve tekrar tekrar uygulamanızı sağlayan en önemli yapılardan biridir. Bir web sitesinde birden fazla butonu, başlığı veya paragrafı aynı renkte ve boyutta yapmak istediğinizde, her birine tek tek stil yazmak yerine bir "class" tanımlar ve bu class'ı ilgili HTML etiketlerine giydirirsiniz.

id kullanımı, CSS ve HTML'de belirli bir elementi benzersiz (biricik) olarak tanımlamak ve sadece o elemente özel stil kuralları uygulamak için kullanılan bir yöntemdir. Class (sınıf) yapısı aynı özelliklerin birden fazla elemente uygulanmasını sağlarken, id yapısı tam tersine "bu sayfada bu elementten sadece bir tane var ve sadece buna özel bir tasarım yapmak istiyorum" dediğiniz durumlar için kullanılır.
