CSS3 | Inline CSS Nasıl Kullanılır #2

Inline CSS (Satır İçi CSS), HTML elementlerinin doğrudan kendi satırı içinde, style özniteliği (attribute) kullanılarak biçimlendirilmesidir. Bir web sayfasını tasarlarken CSS kodlarını ayrı bir dosyaya yazmak yerine, doğrudan etkilemek istediğiniz HTML etiketinin içine yazarsınız.

Bu paragraf inline CSS ile renklendirilmiştir.
